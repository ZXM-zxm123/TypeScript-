# TypeScript-
用 TypeScript 和 Canvas 实现接金币游戏。画布底部有一个篮子（圆形或矩形），用鼠标 X 坐标或左右箭头控制。金币（黄色圆形）、钻石（蓝色星形）、炸弹（黑色圆形）从顶部随机位置生成并自由落体。金币得 10 分，钻石得 30 分，炸弹扣 20 分或损失 1 条命（初始 3 命）。连击系统：连续接住 5 个非炸弹物品后，下一个奖励分数翻倍。游戏限时 60 秒，倒计时结束后显示最终分数。每隔 10 秒下落速度增加。道具：磁铁（自动吸附近物品 5 秒）、时间停止（下落暂停 3 秒）。本地保存最高分。提供三套主题（白天、黑夜、热带）。碰撞检测为矩形/圆形，流畅无卡顿。
